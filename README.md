Artificial Angst

Artificial Angst is a catalogue of fiction generated by, assisted by, or interrogating Artificial Intelligence. It explores the uncanny valley of creativity, asking whether the soul resides in the writer or just in the spellcheck.

This website is built using R and Quarto and hosted on GitHub Pages.

üöÄ Getting Started

To run this website locally on your machine, you need RStudio and Quarto installed.

Clone the repository:

git clone [https://github.com/yourusername/artificial-angst.git](https://github.com/yourusername/artificial-angst.git)


Open the Project:
Open the my_ai_stories.Rproj file in RStudio.

Preview the Site:

Go to the Build tab in the top-right pane.

Click "Render Website".

The site will appear in the Viewer pane or your browser.

Create a new folder inside posts/ (e.g., posts/my-new-story/).

Create a file named index.qmd inside that folder.

Add the YAML header at the top of the file:

---
title: "Your Story Title"
description: "A brief summary for the homepage."
author: "AI Model Name"
date: "YYYY-MM-DD"
categories: [Fiction, Sci-Fi]
image: cover.png
---


Write your story content below the header.

(Optional) Add a cover.png image to the same folder.
